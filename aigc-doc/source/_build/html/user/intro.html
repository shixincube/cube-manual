时信魔方是什么？¶

时信魔方 是面向开发者的实时协作开发框架。帮助开发者快速、高效的在项目中集成实时协作能力。从广义上讲，即时消息、音视频通话、实时白板、文档互传、在线会议都属于实时协作的功能范畴。这些功能帮助人们迅速地传递消息，同步资料并跟踪信息流，从而达到高效协作的目的。
