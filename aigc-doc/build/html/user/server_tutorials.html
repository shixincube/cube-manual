服务器端集成教程¶

在本节中，我们将介绍如何在您的服务器系统里集成时信魔方的服务器。

我们有两种方式来管理和监控服务器，一种是使用时信魔方服务器的管理客户端，另一种是使用 RESTful 方式。使用时信魔方服务器的管理客户端是我们推荐的方式，因为这样能充分利用各个功能的特性达到最佳的集成效果。
