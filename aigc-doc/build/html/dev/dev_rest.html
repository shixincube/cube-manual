文件存储¶

查找文件¶

POST /filestorage/operation/find/¶

按照指定条件查找文件。

Request Headers

  • Content-Type – application/json

Request JSON Object

  • token (string) – 授权联系人的令牌码。

  • md5 (string) – 指定查找文件的 MD5 码。

Response JSON Object

  • data (object) – 查找到的文件标签数据，参看 FileLabel 。

  • name (string) – 文件操作动作。

  • sn (number) – 操作序号。

Status Codes

  • 400 Bad Request – 访问文件服务故障。

  • 403 Forbidden – 参数错误。

  • 404 Not Found – 没有找到指定文件。

上传文件（Binary）¶

POST /filestorage/file/¶

以二进制流方式上传文件数据。

Query Parameters

  • token (string) – 授权联系人的令牌码。

  • filename (string) – 文件名。

  • filesize (number) – 文件大小（ 选填 ）。

  • modified (number) – 文件最后修改的时间戳（ 选填 ）。

  • sn (number) – 操作序号（ 选填 ）。

Response JSON Object

  • data (object) – 上传的文件数据。
    fileName - 文件名。
    fileSize - 文件大小。
    fileCode - 文件码。
    lastModified - 文件最后一次修改时间戳。
    position - 上传数据结束位置。

  • name (string) – 文件操作动作。

  • sn (number) – 操作序号。

{
    "data": {
        "fileName": "yuebing.wav",
        "fileSize": 10248112,
        "fileCode": "UxUiHFebTvyQusIRBQCpANsMdYfWGlHGwVxHlbzAxepzvKBXapMJJZcNkOXiZpEx",
        "lastModified": 1607927012129,
        "position": 10248112
    },
    "name": "uploadFile",
    "sn": 187653391
}
Status Codes

  • 400 Bad Request – 访问文件服务故障。

  • 403 Forbidden – 读取上传数据错误。

  • 404 Not Found – 读取令牌码数据故障。

  • 406 Not Acceptable – 服务不可用。

  • 503 Service Unavailable – 令牌码无效。
