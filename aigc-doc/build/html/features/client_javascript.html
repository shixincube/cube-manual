JavaScript 客户端¶

时信魔方的 Web/HTML5 客户端应用程序是使用时信魔方 JavaScript SDK 开发的即时通信应用，您既可以基于此应用程序进行二次开发，也可以以这个应用程序为参考，了解 JavaScript SDK 的使用。


Web/HTML5 客户端应用程序¶

您可以访问 Gitee 上的 cube-javascript 项目 或者 GitHub 上的 cube-javascript 项目 获取客户端应用程序代码，参考 客户端应用程序 进行编译和安装。


JavaScript SDK¶

时信魔方 JavaScript SDK 提供了使用 SHM 协议访问时信魔方服务器的能力，应用程序使用该 SDK 实现与时信魔方服务器的通信。

API 参考文档 ：时信魔方 JavaScript SDK API 文档.
